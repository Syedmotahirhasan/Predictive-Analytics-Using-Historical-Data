📊 Predictive Analytics Using Historical Data
📌 Project Overview

This project demonstrates how historical sales data can be used to predict future sales using Linear Regression in Python. After building and evaluating the prediction model, an interactive Power BI dashboard was created to visualize sales trends, prediction results, and business insights.

🎯 Objectives
Analyze historical sales data
Build a sales prediction model using Linear Regression
Evaluate model performance
Create an interactive Power BI dashboard
Generate business insights for decision-making
🔹 Project Workflow
Loaded and explored the sales dataset using Pandas
Cleaned the dataset by handling missing values and duplicates
Selected important features for prediction
Split the dataset into training and testing sets
Built a Linear Regression model using Scikit-Learn
Predicted future sales
Evaluated the model using R² Score, MAE, MSE, and RMSE
Visualized Actual vs Predicted Sales using Matplotlib
Exported prediction results to CSV
Built an interactive Power BI dashboard
